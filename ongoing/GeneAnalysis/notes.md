### Hydrolytic Deamination  

C => T  
G => A

cytosine to uracil  C|G -> U|G -> U|A + C|G -> T|A + C|G  
C|G => T|A transition  

5-methyl-cytosine to thymine  
m-C|G -> T|G -> T|A + C|G  
C|G => T|A transition

### Oxidation

G => T  
C => A  
G|C -> 8-oxo-G|C -> 8-oxo-G|A & G|C -> T|A & G|C  
G|C => T|A transversion
